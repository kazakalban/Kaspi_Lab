Практика 3

1. Создать функцию, которая:
- производит чистку данных в таблице customer_orders
- чистые данные помещает в таблицу cleaned_customer_orders
- данные, которые невозможно пофиксить, поместить в таблицу problem_customer_orders с описанием причины
- возвращает количество проблемных записей

2. Запустить следующий скрипт:

create table total_spent(
    customer_name text,
    spent real
);

insert into total_spent values
        ('John Smith', 1300),
        ('Mary Johnson', 430.28);

3. Создать хранимую процедуру, которая:
- в качестве параметра принимает число N
- делает выборку из таблицы cleaned_customer_orders еще необработанных N записей
- производит подсчет суммы покупок по клиентам
- обновляет таблицу total_spent
